CSCE 242 - Web Applications

Nicholas Kinard

Description: This class teaches students to utilize HTML and CSS to create the front-end of a website. Students will also use JavaScript to allow for interactions and server side scripting to pull data from a server.

Assignments

Assignment 1 - Basic HTML

This assignment had me create a basic website understanding the basics of HTML.

Assignment 2 - Basic CSS

This assignment had me begin using CSS to style this main page as well as modify a webpage to our liking.

Assignment 3 - Page Layout

This assignment had me use CSS to explore Flexboxes and utilize media to have designs that work for many resolutions.

Assignment 4 - CSS Page

This assignment had me recreate a USC website page to practice my CSS skills.

Assignment 5

This assignment had me practice basic skills in JavaScript creating a hello box, a color picker, and an image generator.

Assignment 6 - JS Conditionals

This assignment had me practice JS skills and use conditional if statements.

Assignment 7 - Loops

This assignment had me practice JS skills and use for loops and setInterval.

Assignment 8 - Arrays

This assignment had me practice JS skills and use an array and an associative array to fill in information.

Assignment 9 - Classes

This assignment had me practice JS skills by using classes and populating an HTML file through JS functions.
